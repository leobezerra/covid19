Sobre o projeto

Os dashboards de análise da COVID-19 disponibilizados neste site são um projeto colaborativo organizado por Leonardo Bezerra, junto à comunidade de ciência de dados do Instituto Metrópole Digital, do Rio Grande do Norte e do Brasil.

Para contribuir, entre em contato por um dos seguintes canais:

  • Email: leobezerra@imd.ufrn.br
  • Telegram: @leo_bezerra

Origem dos dados

Os dados disponíveis nesta página foram obtidos de diferentes fontes.

Dados mundiais

  • COVID-19: John Hopkins CSSE
  • População mundial: Penn World Table

Dados do Brasil

  • COVID-19 e população estimada: brasil.io
  • Dados georeferenciados: IBGE

Dados do RN

Os dados da COVID-19 RN foram extraídos dos boletins epidemiológicos divulgados pela SESAP-RN. A extração deve ser creditada a:

  • brasil.io: casos confirmados até 24/03/2020
  • vitor-saldanha: casos suspeitos até 24/03/2020
  • leobezerra: latitude e longitude dos municípios e dados posteriores a 24/03/2020

A partir do dia 29/03/2020, a extração passou a ser feita de forma automatizada, utilizando o scraper que deve ser creditado a:

  • gabicavalcante: autoria
  • leobezerra: revisão e validação

Os dados georeferenciados foram obtidos do IBGE e do ArcGis.
